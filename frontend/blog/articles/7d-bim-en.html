1 Module 1: Understanding 7D BIM

1.1 What is 7D BIM?

7D BIM extends the BIM model for facility management and operations. It transforms construction data into an operational asset database that FM teams use throughout the building's lifecycle.

üîë Key Formula

7D BIM = 6D BIM + Facility Management & Operations

1.2 The Building Lifecycle

A building spends:

  • 1-3 years in Design & Construction
  • 30-50+ years in Operations

7D BIM ensures that BIM investment continues delivering value during the longest phase of a building's life.

1.3 7D BIM Capabilities

Capability Description Example
Asset Tracking Manage all equipment data HVAC units, elevators, pumps
Preventive Maintenance Scheduled maintenance plans Filter changes, inspections
Space Management Track room usage and allocation Occupancy, move planning
Work Orders Manage maintenance requests Repair tickets, service calls
Emergency Planning Access building data for response Fire routes, shut-off valves
Warranty Tracking Monitor equipment warranties Claim before expiration

The #1 reason FM handovers fail is starting too late. Begin FM data collection at design stage, not at substantial completion!

3 Module 3: COBie Data Standard

3.1 What is COBie?

üìä COBie (Construction Operations Building Information Exchange)

A standardized spreadsheet format (or IFC extension) for delivering building data to owners/FM teams. It extracts non-graphical data from BIM models into a structured format.

3.2 COBie Worksheets

Worksheet Contains Example Data
Facility Building information Name, address, area
Floor Level information Floor names, elevations
Space Room data Room names, areas, usage
Type Equipment types Manufacturer, model, specs
Component Individual assets Serial number, install date
System Systems grouping HVAC Zone 1, Fire System A
Job Maintenance tasks Annual inspection, filter change
Resource Spare parts, materials Replacement filters, belts
Document Linked documents O&M manuals, warranties

3.3 COBie in Revit

To export COBie from Revit:

  1. Install COBie Extension for Revit
  2. Populate required parameters during modeling
  3. Map Revit parameters to COBie fields
  4. Export to COBie spreadsheet or IFC with COBie
  5. Validate using COBie QC tools

5 Module 5: Asset Management

5.1 Asset Data Requirements

Critical data for each maintainable asset:

Category Data Fields
Identification Tag number, serial number, barcode/QR
Location Building, floor, room, coordinates
Specifications Manufacturer, model, capacity, power
Dates Install date, warranty expiry, expected life
Maintenance PM schedule, service history, spare parts
Documents O&M manual, specifications, warranties

5.2 Maintenance Types

  • Reactive (Corrective): Fix when broken - most expensive
  • Preventive (Scheduled): Regular maintenance per schedule
  • Predictive (Condition-based): Maintain based on sensor data
  • Prescriptive: AI recommends optimal maintenance
Moving from Reactive to Preventive maintenance can reduce maintenance costs by 12-18%. Moving to Predictive adds another 8-12% savings.
