üìÑ 1. What is IFC?

IFC (Industry Foundation Classes) is an open, neutral standard for exchanging BIM data between different software. It's managed by buildingSMART International.

1.1 Definition:

IFC = "Common language" for communication between different BIM software without data loss.

1.2 IFC Characteristics:

  • Open: Not owned by any company
  • Neutral: Works with all software
  • Standardized: ISO 16739
  • Extensible: Supports future development
