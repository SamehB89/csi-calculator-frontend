Cost Breakdown Structure (CBS) in Construction

January 7, 2026 10 min read

üìä What is Cost Breakdown Structure (CBS)?

Cost Breakdown Structure (CBS) is a comprehensive hierarchical classification of all project costs. While WBS focuses on "what we will build", CBS focuses on "how much everything will cost". Both are complementary and indispensable.

üîó Relationship Between CBS and WBS

Think of CBS as a financial layer parallel to WBS. Each work package in WBS should have a corresponding cost code in CBS:

  • WBS: Answers "What?" - What are the activities and deliverables?
  • CBS: Answers "How much?" - What is the cost of each element?

üìÅ Typical CBS Structure

CBS is organized in hierarchical levels:

Level 1: Main Classification

  • 01 - Structural Works
  • 02 - Architectural Works
  • 03 - Electrical Works
  • 04 - Mechanical Works
  • 05 - Site Works

Level 2: Details

Example for Structural Works (01):

  • 01.01 - Excavation Works
  • 01.02 - Concrete Works
  • 01.03 - Reinforcement Works
  • 01.04 - Formwork

Level 3: Fine Details

Example for Concrete Works (01.02):

  • 01.02.01 - Foundation Concrete
  • 01.02.02 - Column Concrete
  • 01.02.03 - Slab Concrete
  • 01.02.04 - Beam Concrete

üè∑Ô∏è Cost Coding System

Good coding ensures easy tracking and reporting. Example coding system:

XX.YY.ZZ.NNN
XX = Main Division (e.g., 01 for Structural)
YY = Sub-Division (e.g., 02 for Concrete)
ZZ = Type (e.g., 03 for Slabs)
NNN = Sequential Number

üìà Benefits of Organized CBS

  • Accurate Tracking: Know where every dollar goes.
  • Comparisons: Easily compare actual vs planned costs.
  • Reporting: Generate financial reports at any detail level.
  • Integration: Link CBS with accounting systems and ERP.
  • Lessons Learned: Build historical pricing database.

üí° Implementation Tips

  • Use a unified coding system across all company projects.
  • Don't make CBS too complex - 4 levels usually suffice.
  • Ensure CBS aligns with WBS for easy integration.
  • Train accounting team on using correct codes.
